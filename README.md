# Model-Comparison-of-Machine-Learning-Algorithms-on-Iris-Dataset
APPROACH
I used Google Colab for this model comparison
First, I loaded the Iris dataset. 
Then I split the data into training and testing parts.
After that I used three models which are:
1.	Logistic Regression, 
2.	KNN, and 
3.	Decision Tree. 
I trained each model on training data and then tested it on test data. Finally, I compared their accuracy.

DIFFICULTIES FACED
This whole machine learning thing was new for me. At first I didn’t even understand how to use sklearn library and didn’t know how models are trained. I was also confused about how to split the data and check accuracy.

RESOLUTION OF DIFFICULTIES
I referred to online resources and videos on youtube. Even asked AI like ChatGPT for basic knowledge. After seeing some basic code, I started understanding how to use fit and predict functions. Then it became easier to implement the models.

RESULT
All three models worked properly and gave good accuracy
Logistic Regression Accuracy: 0.9667
KNN Accuracy: 0.9333
Decision Tree Accuracy: 0.9333
Logistic Regression model worked best for iris dataset.
